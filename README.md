# Philosophy Geo - 哲学家时空地图

一个交互式可视化应用，展示全球哲学家的地理分布、时间脉络与思想传承。

## 简介

Philosophy Geo 是一个基于 Vue 3 + TypeScript + Leaflet 构建的 Web 应用，通过地图、时间轴和网络关系图，帮助用户探索从古希腊到现代的哲学家生平、学派归属与思想影响。

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **TypeScript** - 类型安全的 JavaScript 超集
- **Vite** - 下一代前端构建工具
- **Leaflet** - 开源交互式地图库
- **Pinia** - Vue 状态管理方案

## 功能列表

### 地图功能
- 交互式世界地图展示哲学家出生地
- 按学派/地区筛选标记
- 点击标记查看哲学家详情
- 地图缩放与平移

### 时间轴功能
- 可拖拽时间范围滑块（公元前600年 - 公元2000年）
- 实时过滤显示对应时期的哲学家
- 时间轴与地图联动

### 搜索与筛选
- 按姓名搜索哲学家
- 按学派筛选（儒家、道家、柏拉图主义等）
- 按地区筛选（古希腊、中国、印度、欧洲等）
- 按重要性排序

### 详情面板
- 哲学家生平信息
- 思想主张列表
- 师承关系展示
- 多语言名称显示

## 开发命令

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 预览生产构建
npm run preview
```

## 构建命令

```bash
# 类型检查并构建生产版本
npm run build
```

构建产物将输出到 `dist/` 目录。

## 数据说明

本项目包含 20 位重要哲学家的数据：

### 古希腊（5位）
- 苏格拉底、柏拉图、亚里士多德、伊壁鸠鲁、塞内卡

### 中国（5位）
- 孔子、老子、庄子、孟子、荀子

### 印度（2位）
- 佛陀、龙树

### 阿拉伯（1位）
- 伊本·西那（阿维森纳）

### 近代欧洲（3位）
- 笛卡尔、康德、黑格尔

### 现代（4位）
- 维特根斯坦、海德格尔、胡塞尔、罗素

每位哲学家包含以下信息：
- 多语言姓名（中文、英文、原文）
- 生卒年份与地点（含地理坐标）
- 所属学派
- 师承关系
- 核心思想
- 重要性评分
- 生平简介
