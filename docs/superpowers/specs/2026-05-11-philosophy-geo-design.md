# 哲学家历史地理可视化平台 - 设计文档

**日期**: 2026-05-11  
**类型**: 单页应用 (SPA)  
**状态**: 设计完成，待实施

---

## 1. 项目概述

### 1.1 核心定位
数据可视化导向的哲学史探索平台，展示公元前800年至2020年代全球哲学思想的地理分布、时间演变和传播路径。

### 1.2 目标用户
- 哲学研究者（寻找地理维度的研究视角）
- 哲学爱好者（直观理解哲学传统的空间关系）
- 教育工作者（教学演示工具）
- 普通知识大众（哲学史入门）

### 1.3 成功标准
- 能够清晰展示不同时期哲学活动的地理重心转移
- 用户可通过多维度筛选快速定位感兴趣的哲学家/学派
- 故事线模式能有效传达哲学传统的传承与演变

## 2. 技术架构

### 2.1 技术栈
| 层级 | 技术 | 用途 |
|------|------|------|
| 框架 | Vue 3 + TypeScript | 核心应用框架，组合式API |
| 构建 | Vite | 快速开发与构建 |

### 2.2 地图底图
- 使用 CartoDB Dark Matter 或类似深色主题底图

- 按需加载地理边界数据（世界/区域/城市）

### 2.3 数据存储
- 初期：JSON 静态文件（版本控制）
- 未来可扩展：REST API 或 GraphQL

## 3. 数据模型

### 3.1 核心实体

```typescript
// 哲学家
interface Philosopher {
  id: string;
  name: {
    zh: string;      // 中文名
    en: string;      // 英文名
    original?: string; // 原文名（希腊/阿拉伯/梵文等）
  };
  birth: {
    year: number;    // 公元前为负数
    location: GeoPoint;
    placeName: string;
  };
  death?: {
    year: number;
    location?: GeoPoint;
    placeName?: string;
  };
  schools: string[];       // 学派ID列表
  influences: string[];    // 影响的哲学家ID
  influencedBy: string[];  // 受谁影响的哲学家ID
  works: Work[];
  ideas: string[];         // 核心概念标签
  travels?: Travel[];      // 生平游历
  importance: number;      // 影响力权重 (1-10)
}

// 地理位置
interface GeoPoint {
  lat: number;
  lng: number;
}

// 游历记录
interface Travel {
  from: GeoPoint;
  to: GeoPoint;
  yearStart: number;
  yearEnd?: number;
  reason: string;          // 游历原因
}

// 著作
interface Work {
  title: { zh: string; en: string };
  year: number;
  importance: number;
}

// 学派
interface School {
  id: string;
  name: { zh: string; en: string };
  region: Region;          // 地理大区域
  period: { start: number; end: number };
  color: string;           // 地图标记颜色 (HEX)
  description: string;
  coreIdeas: string[];
}

// 地理区域枚举
enum Region {
  GREECE = '古希腊',
  ROME = '古罗马',
  EUROPE_WEST = '西欧',
  EUROPE_CENTRAL = '中欧',
  EUROPE_EAST = '东欧',
  CHINA = '中国',
  JAPAN = '日本',
  INDIA = '印度',
  PERSIA = '波斯/伊朗',
  ARABIA = '阿拉伯世界',
  AMERICAS = '美洲',
  OTHER = '其他'
}

// 故事线
interface StoryLine {
  id: string;
  title: { zh: string; en: string };
  description: string;
  period: { start: number; end: number };
  waypoints: Waypoint[];
  philosophers: string[];  // 相关哲学家ID
  color: string;
}

// 故事线路点
interface Waypoint {
  location: GeoPoint;
  placeName: string;
  year: number;
  philosopherId?: string;
  description: string;
}
```

### 3.2 初始数据集

**哲学家（MVP阶段：20位核心人物）**
- 古希腊：苏格拉底、柏拉图、亚里士多德、伊壁鸠鲁、芝诺
- 中国：孔子、老子、孟子、庄子、荀子
- 印度：佛陀、龙树、商羯罗
- 阿拉伯：阿尔肯迪、阿维森纳、阿威罗伊
- 近代欧洲：笛卡尔、康德、黑格尔
- 现代：维特根斯坦、海德格尔

**学派（MVP阶段：10个核心学派）**
- 柏拉图学院、亚里士多德学派、伊壁鸠鲁学派、斯多葛学派
- 儒家、道家、墨家、法家
- 瑜伽行派、吠檀多派
- 逍遥学派、阿维森纳主义
- 德国观念论、分析哲学、现象学

**故事线（第二阶段：5条预设路线）**
1. 苏格拉底之路：雅典学派的传承
2. 禅宗东渐：印度→中国→日本
3. 启蒙运动：英国经验论→法国启蒙→德国古典哲学
4. 现象学运动：布伦塔诺→胡塞尔→海德格尔
5. 分析哲学源流：弗雷格→罗素→维特根斯坦

## 4. 界面设计

### 4.1 整体布局

```
┌─────────────────────────────────────────┐
│  Logo    [🔍 全局搜索...]    视图切换按钮  │  ← 顶部导航栏 (60px)
├──────────┬──────────────────────────────┤
│          │                              │
│  侧边栏   │        地图主视图            │
│  筛选器   │      （Leaflet 容器）        │
│  + 列表   │                              │
│          │                              │
│ (280px)  │         (自适应剩余空间)        │
│          │                              │
├──────────┴──────────────────────────────┤
│  底部时间轴（可收起）                      │  ← 时间轴区域 (80px, 可收起)
└─────────────────────────────────────────┘
```

### 4.2 顶部导航栏

**元素：**
- 左侧：Logo + 项目名称"Philosophy Geo"
- 中央：全局搜索框
  - Placeholder: "搜索哲学家、学派、地点..."
  - 快捷键: `Cmd/Ctrl + K`
  - 实时下拉建议（哲学家、学派、地点）
  - 点击结果：定位地图 + 打开详情
- 右侧：视图切换按钮组
  - 地图视图（默认）
  - 关系网络视图
  - 故事线模式

### 4.3 侧边栏

**筛选面板：**
- 地域筛选（多选）：欧洲、东亚、南亚、阿拉伯、其他
- 学派筛选（多选）：下拉展开学派列表
- 时代快速跳转：古代(-800~500)、中世纪(500~1500)、近代(1500~1900)、现代(1900~)
- 主题标签（可选）：形而上学、伦理学、政治哲学、认识论、逻辑学、美学
- 重置按钮

**哲学家列表：**
- 按影响力排序
- 每项显示：颜色圆点（学派）+ 姓名 + 生卒年
- 点击：地图定位并高亮
- Hover：显示迷你信息卡

### 4.4 地图主视图

**图层层级（从底到顶）：**
1. 深色底图（CartoDB Dark Matter）
2. 区域高亮层（重要哲学中心）
3. 热力图层（哲学活动密度，可开关）
4. 轨迹线层（思想传播路径，可开关）
5. 哲学家标记层

**标记设计：**
- 圆形标记，直径 = 20px + importance * 3
- 颜色 = 学派主色
- 边框：白色2px
- Hover：放大1.2倍 + 显示姓名标签
- 点击：弹出详情卡片

**详情卡片内容：**
- 头像/姓名/生卒年
- 学派标签
- 一句话简介
- 核心概念标签
- "查看详情"按钮（展开完整信息面板）

### 4.5 底部时间轴

**功能：**
- 可拖动时间范围选择（双滑块）
- 缩放控制：世纪 / 年代 / 年份
- 播放/暂停按钮（自动演进历史）
- 速度控制：1x / 2x / 5x
- 收起/展开按钮

**视觉：**
- 半透明深色背景（rgba(30,30,30,0.9)）
- 时间刻度清晰可见
- 当前选中范围高亮显示

### 4.6 关系网络视图

**布局：**
- 全屏D3力导向图
- 节点 = 哲学家（大小 = importance）
- 边 = 影响关系（粗细 = 影响程度）
- 节点颜色 = 学派

**交互：**
- 拖拽节点调整布局
- 滚轮缩放
- 点击节点：高亮相关连接
- 图例说明

### 4.7 故事线模式

**布局：**
- 左侧：故事线选择列表
- 主视图：地图 + 动画路线

**播放器控制：**
- 播放/暂停/前进/后退
- 进度条显示当前位置
- 章节标记点

**视觉效果：**
- 路线以动画虚线展示
- 当前节点脉冲高亮
- 信息面板显示当前位置的故事描述

- 中文：系统默认（苹方/PingFang SC、微软雅黑）

- 标题：18-24px
- 正文：14-16px
- 辅助信息：12px

### 5.3 组件风格

- 玻璃拟态效果（Glassmorphism）
- 圆角：8px（小元素）、12px（面板）
- 阴影：柔和投影增加层次感
- 过渡动画：200-300ms ease

## 7. 性能考虑

### 7.1 数据加载

- 初始加载核心数据集（< 100KB）
- 图片懒加载（哲学家头像）
- 地图瓦片按需加载

### 7.2 渲染优化

- 标记聚类（Marker Clustering）处理密集区域
- 虚拟滚动处理长列表
- 防抖处理筛选操作

### 7.3 构建优化

- 代码分割（按视图）
- 资源预加载关键路径
- Gzip/Brotli 压缩

## 9. 风险与应对

| 风险 | 可能性 | 影响 | 应对策略 |
|------|--------|------|----------|
| 数据准确性争议 | 高 | 中 | 标注数据来源，提供反馈机制 |
| 地理坐标数据缺失 | 中 | 中 | 允许近似位置，标注不确定性 |
| 东西方哲学平衡 | 中 | 高 | 确保数据集代表性的审查机制 |
| 性能问题（大数据量） | 中 | 中 | 实现懒加载和虚拟化 |
